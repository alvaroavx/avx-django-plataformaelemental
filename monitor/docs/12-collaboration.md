# Colaboracion entre perfiles

## Senior Django developer
Responsable de:
- modelos
- servicios
- forms
- vistas
- tests backend
- migraciones

## Responsive web designer
Responsable de:
- estructura visual
- templates
- estados vacios
- mobile
- accesibilidad basica
- consistencia AVX Mission Control

## QA analyst
Responsable de:
- escenarios de prueba
- riesgos
- regresiones
- criterios de aceptacion
- cobertura manual y automatizada

## Regla de coordinacion
Cada tarea debe indicar:
- objetivo
- archivos esperados
- criterio de aceptacion
- validacion requerida

Si hay conflicto entre estetica y operacion, gana la claridad operativa.
