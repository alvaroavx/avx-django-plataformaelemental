# Pagina publica

## Estado
Fuera de fase 1.

## Objetivo futuro
Exponer una pagina publica opcional con estado resumido de sitios o servicios.

## Reglas futuras
- Debe ser opt-in por proyecto o sitio.
- No debe exponer errores internos sensibles.
- Debe permitir desactivar indexacion si corresponde.
- Debe tener cache o consultas livianas.

## Contenido candidato
- Estado general.
- Ultima actualizacion.
- Incidentes activos.
- Historial resumido.
