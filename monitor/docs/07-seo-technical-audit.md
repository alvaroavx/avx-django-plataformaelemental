# Auditoria SEO y tecnica

## Objetivo
Entregar senales tecnicas basicas para entender la salud de un sitio.

## Fase 1
Discovery inicial:
- Estado HTTP.
- URL final.
- Titulo.
- Meta description.
- Tiempo de respuesta.
- SSL valido si es posible.

## Fase 2
Auditoria extendida:
- H1 presente.
- Canonical.
- Robots meta.
- Robots.txt.
- Sitemap.xml.
- Open Graph basico.
- Peso aproximado de respuesta.

## Reglas
- El scoring SEO no entra hasta que existan datos historicos confiables.
- Cada metrica debe documentar fuente y criterio.
- No bloquear la creacion del sitio por fallas SEO.
