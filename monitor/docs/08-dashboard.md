# Dashboard general

## Objetivo
Mostrar estado rapido de los sitios monitoreados.

## Fase 1
Debe mostrar:
- Total de sitios.
- Sitios activos.
- Sitios con error.
- Ultimos sitios agregados.
- Accion principal para agregar sitio.

## Layout
Desktop:
- Cards resumen arriba.
- Listado o tabla de sitios abajo.
- Acciones visibles sin abrir menus profundos.

Mobile:
- Cards apiladas.
- Lista de sitios en formato compacto.
- Accion principal visible.
- Evitar tablas que requieran zoom.

## Estados vacios
Si no existen sitios, mostrar:
- mensaje breve
- boton para agregar sitio
- sin graficos decorativos innecesarios
